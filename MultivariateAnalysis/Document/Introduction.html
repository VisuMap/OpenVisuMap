Introduction
The MultivaraiteAnalysis plugin extends VisuMap with services for variouse type of  multivariate data analysis. Currently the plugin provides the following services:
  • Linear Discreminate Analysis (LDA).
  • Partial Least Squares (PLS) Analysis.
  • Cannonic Correlation Analysis (CCA).
All three services are linear data analysis with so called projection pursuit strategy: they try to find limited number of directions in the high dimensional space, so that the projection of a data set to these directions will maximize certain properties. So for instance, the LDA method tries to find a set of directions, so that different types of data points will have maximal separatation after the projection. Figure 1 illustrates how LDA find a direction that maximally separates two types of data points. Notice that the method of princimpal component analysis (PCA) is also a projection pursuit method, as it finds directions that maximize the variance of the whole dataset.



Fingure 1: LDA vs. PCA in projecting 2-dimensional data to 1-dimensional space: LDA maximaizes the separation whereas PCA maximizes the variance.


PLS method provides basically opposite service as LDA: whereas LDA tries to find features (rotation direction) to distinguish groups of data points; PLS tries to find common features between data groups. More particularily, for two given data points set X and Y in a high dimensional space, PLS algorithm finds TWO sets of directions, so that the projection of X and Y to these two sets of directions will have maximal covariance with each other. Figure 2 illustrates how PLS finds a direction for two data point sets, so that the projection of the two sets will have maximal covariance with each other.




Fingure 2: PLS finds two directions for two data clusters, so that the projections of the clusters to the directions have maximal covariance to each other.

The CCA method works very similarly as PLS with the exception that CCA maximizes the correlation  of the projected data instead of their covariance.  Since, unlike the covariance, the correlation is in general independent on the scaling range of the data, CCA can be applied to data set with different scaling range ( by ignoring the scaling information). On the otherside, PLS will preserve the scaling information and is therefore normally only appropriate to explore data with common scaling range. It should be pointed out that  CCA, as projection method, does not perserve the scaling, since its projection base vectors are normally not orthogonal.

As common to all projection prusit methods, directions found by LDA, PLS are orthogonal unit vectors, and each direction is associated with a factor,  called the eigenvalue, that indicates how much that direction contributes to the to-be maximized property. In practices, it is often sufficient to explore just few directions with the largest eigenvalues. Becouse of this characteristics, these methods are sometimes used as dimensionality reduction methods.

 Using the MultivariateAnalysis plugin

The MultivariateAnalysis plugin is delivered as a zipped directory file MultivariateAnalysis.zip.  The plugin is installed as following:
  1. Decompress MultivariateAnalysis.zip.
  2. Start VisuMap application.
  3. Press F5 in VisuMap; then, when prompt for a script file, select the file SetupMva.js.
After the installation, the plugin will offer its service through the plugin object named "MultivariateAnalysis". The user can obtain a reference to this object in a script by the statement "var mva = vv.FindPluginObject("MultivariateAnalysis");". Please see the chapter "VisuMap.MultiVariateAnalysis NameSpace" of this document for more information about how to use this plugin object.

For the sake of convienence the installation procedure also installs three context menus to enable users to do services on selected data. The three context menus are as following:

  • Do LDA: Perform LDA analysis on selected data. The selected data points should have at least two different types. This operation will display the projected data in a 3D view window.  You can perform binary LDA by marking some data points as highlighted (through the common context menu), then select another set of data points, and then choose this context menu. When doing binary LDA, the algorithm will just find a direction that maximally separates the two selected data point sets.

  • Do PLS:  Perform PLS analysis. In order to use this service, first mark a set data points as highlighted (through the common context menu); then selec another set of data points; then choose this context mneu.  This operation will display the projections of the two selected data point sets in two value diagram windows.

  • Do CCA: Perform CCA analysis. In order to use this service, first marks a set of data points as highlighted (through the common context menu); then select another set of data points; then choose this context mneu.  This operation will display the projections of the two selected data point sets in two value diagram windows.

In order to uninstall the MultivariateAnalysis plugin just press F5 in VisuMap, then choose the script UninstallMva.js.
